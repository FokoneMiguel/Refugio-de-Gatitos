## 3. 📧 RECEVOIR LES MESSAGES DU FORMULAIRE DANS TA BOÎTE MAIL {#emailjs}

On utilise **EmailJS** — service gratuit jusqu'à 200 emails/mois, aucun serveur requis.

---

### ÉTAPE 1 — Créer ton compte EmailJS

1. Ouvre ton navigateur et va sur **https://www.emailjs.com**
2. Clique sur le bouton **"Sign Up Free"** (en haut à droite)
3. Remplis le formulaire avec :
   - Ton adresse email (où tu veux recevoir les messages des clients)
   - Un mot de passe
4. Vérifie ton email → clique sur le lien de confirmation

---

### ÉTAPE 2 — Connecter ton compte Gmail (ou autre)

1. Une fois connecté à EmailJS, dans le menu gauche clique sur **"Email Services"**
2. Clique sur le bouton **"Add New Service"**
3. Une liste de services apparaît — clique sur **"Gmail"**
4. Clique sur **"Connect Account"**
5. Une fenêtre Google s'ouvre → connecte-toi avec le compte Gmail où tu veux recevoir les messages
6. Autorise les permissions
7. EmailJS te ramène au tableau de bord
8. Tu vois maintenant ton service avec un **Service ID** (exemple : `service_xxxxxxx`)
9. **📋 Copie et note ce Service ID quelque part**

---

### ÉTAPE 3 — Créer le modèle d'email

1. Dans le menu gauche, clique sur **"Email Templates"**
2. Clique sur **"Create New Template"**
3. L'éditeur s'ouvre. Configure chaque champ comme suit :

**Champ "To Email" :** tape ton adresse Gmail (celle qui va recevoir les messages)

**Champ "From Name" :** `{{from_name}}`

**Champ "Subject" (objet du mail) :**
```
🐱 Nueva solicitud — {{from_name}} quiere {{breed}}
```

**Zone de contenu (Content/Body) — copie exactement ceci :**
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  NUEVA SOLICITUD DE ADOPCIÓN
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

👤 Nombre:        {{from_name}}
📧 Email:         {{from_email}}
📞 Teléfono:      {{phone}}
🐱 Raza deseada:  {{breed}}

💬 Mensaje del cliente:
─────────────────────
{{message}}
─────────────────────

Responde directamente a: {{reply_to}}

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Refugio de Gatitos — Formulario Web
```

4. Clique sur **"Save"** (bouton bleu en haut à droite)
5. **📋 Note le Template ID** qui s'affiche (exemple : `template_xxxxxxx`)

---

### ÉTAPE 4 — Récupérer ta clé publique

1. Dans le menu gauche, clique sur **"Account"**
2. Clique sur l'onglet **"General"**
3. Tu vois une section **"Public Key"**
4. **📋 Copie cette clé** (exemple : `aBcDeFgHiJkLmNoP`)

---

### ÉTAPE 5 — Coller tes clés dans le fichier HTML

Ouvre ton fichier `index.html` dans un éditeur de texte.

> Sur Windows : **Notepad++** (gratuit, télécharge sur notepad-plus-plus.org)
> Sur Mac : **TextEdit** ou **VS Code**
> Sur Android : **Acode** (application gratuite sur Play Store)

Cherche ces 3 lignes avec **Ctrl+F** (ou Cmd+F sur Mac) en tapant `TU_SERVICE_ID` :

```javascript
const EMAILJS_SERVICE_ID  = 'TU_SERVICE_ID';
const EMAILJS_TEMPLATE_ID = 'TU_TEMPLATE_ID';
const EMAILJS_PUBLIC_KEY  = 'TU_PUBLIC_KEY';
```

Remplace les valeurs par les tiennes. Exemple :

```javascript
const EMAILJS_SERVICE_ID  = 'service_ab12cd34';
const EMAILJS_TEMPLATE_ID = 'template_ef56gh78';
const EMAILJS_PUBLIC_KEY  = 'aBcDeFgHiJkLmNoP';
```

Enregistre le fichier (**Ctrl+S**).

## 4. 📱 BOUTON WHATSAPP — DIRIGER VERS TA CONVERSATION {#whatsapp}

### Comment ça marche

Quand un client clique sur le bouton vert WhatsApp de ton site, il est redirigé directement vers une conversation WhatsApp avec **ton numéro**, avec un **message pré-rempli**.

### Trouver la ligne à modifier

Dans ton fichier `index.html`, cherche (Ctrl+F) :
```
TU_NUMERO_WHATSAPP
```

Tu verras cette ligne :
```html
<a href="https://wa.me/TU_NUMERO_WHATSAPP?text=Hola%2C%20me%20interesa...
```

### Remplacer par ton numéro

Ton numéro doit être écrit **sans espaces, sans le signe +, avec l'indicatif pays** :

| Ton numéro affiché | Ce que tu écris dans le code |
|---|---|
| +237 691 23 45 67 | `237691234567` |
| +237 677 89 01 23 | `237677890123` |
| +34 607 36 05 29 | `34607360529` |

**Exemple final :**
```html
<a href="https://wa.me/237691234567?text=Hola%2C%20me%20interesa%20adoptar%20un%20gatito.%20%C2%BFPodr%C3%ADa%20darme%20m%C3%A1s%20informaci%C3%B3n%3F"
```

### Personnaliser le message automatique

Le message `Hola%2C%20me%20interesa%20adoptar%20un%20gatito...` est du texte encodé. En clair ça dit : *"Hola, me interesa adoptar un gatito. ¿Podría darme más información?"*

Pour changer ce message :
1. Va sur **https://www.urlencoder.org**
2. Dans la zone de texte, écris ton message en español (ou français)
3. Clique sur **"Encode"**
4. Copie le résultat
5. Remplace la partie après `?text=` dans ton code par ce que tu viens de copier

**Exemple de message personnalisé :**
Texte original : `Bonjour, je suis intéressé par un de vos chatons. Pouvez-vous me donner plus d'informations ?`
Encodé : `Bonjour%2C%20je%20suis%20int%C3%A9ress%C3%A9%20par%20un%20de%20vos%20chatons...`

### Tester le bouton WhatsApp

1. Ouvre ton site sur ton téléphone Android/iPhone
2. Fais défiler vers le bas jusqu'à voir le bouton vert WhatsApp apparaître
3. Clique dessus
4. WhatsApp doit s'ouvrir sur ton téléphone avec **ta propre conversation** pré-remplie
5. Si ça ouvre la conversation avec ton numéro → ✅ ça marche !

> 💡 **Conseil** : Utilise **WhatsApp Business** (gratuit) plutôt que WhatsApp normal. Il te permet d'avoir un profil d'entreprise, des réponses automatiques, et des statistiques de messages.

## 5. 📢 GOOGLE ADS — FAIRE DE LA PUBLICITÉ SUR TON SITE {#googleads}

Google Ads te permet d'apparaître en **première position sur Google** quand quelqu'un cherche "gatitos ragdoll precio" ou "comprar gatito maine coon".

### ÉTAPE 1 — Créer ton compte Google Ads

1. Va sur **https://ads.google.com**
2. Clique sur **"Commencer maintenant"**
3. Connecte-toi avec ton compte Gmail
4. Google va te demander ton objectif → choisis **"Obtenir plus de contacts"** (Leads)

### ÉTAPE 2 — Créer ta première campagne

**Paramètres importants :**

| Paramètre | Ce que tu choisis |
|---|---|
| Type de campagne | **Réseau de Recherche** (Search) |
| Objectif | **Prospects / Contacts** |
| Pays cible | Espagne, France (ou le pays où tu livres) |
| Langue | Español + Français + English |
| Budget journalier | 5€ à 10€ pour commencer |

### ÉTAPE 3 — Mots-clés à utiliser

Copie-colle ces mots-clés dans ta campagne (un par ligne) :

```
"comprar gatito maine coon"
"gatitos ragdoll en venta"
"criadero de gatitos"
"gatitos de raza precio"
"comprar gato de raza"
"gatitos british shorthair"
"gatito bengalí precio"
"adoptar gatito persa"
"gatitos fold escocés"
"criadero gatos certificado"
```

**Mots-clés NÉGATIFS (pour ne pas gaspiller ton argent) :**
```
-gratis
-gratuito
-refugio
-rescate
-abandonado
-adopción gratuita
-shelter
```

### ÉTAPE 4 — Rédiger tes annonces

Voici des annonces prêtes à copier-coller :

**Annonce 1 :**
```

Titre 2 : Maine Coon, Ragdoll, Persa, Bengalí
```

### ÉTAPE 5 — Lier ton site à Google Ads

Pour que Google sache quand quelqu'un vient de ton annonce et remplit ton formulaire :

1. Dans Google Ads → clique sur **"Outils et paramètres"** (icône clé en haut)
2. Clique sur **"Conversions"**
3. Clique sur **"Nouvelle action de conversion"**
4. Choisis **"Site Web"**
5. Donne un nom : `Formulaire_Contact`
6. Choisis la catégorie : **"Prospect"**
7. Google te donne un petit code → copie-le
8. Dans ton fichier `index.html`, cherche `</head>` et colle le code juste avant

Google Ads accepte :
